# Mazha Vault — Claude Context Index

This vault stores Hugo's personal context so Claude can be fetched up to speed at the start of any session.

## How to Use with Claude
Paste one of these raw GitHub URLs into your message:

```
https://raw.githubusercontent.com/Mazha96/mazha-vault/main/_context/Hugo_Profile.md
https://raw.githubusercontent.com/Mazha96/mazha-vault/main/_context/Current_Projects.md
```

Then say: **"Fetch my context files and get up to speed."**

## Vault Structure

```
mazha-vault/

│   ├── Hugo_Profile.md        ← identity, work, interests, pets

```
